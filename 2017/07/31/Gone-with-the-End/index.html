Gone with the End

Posted on 2017-07-31 | Edited on 2018-07-16

Margaret Mitchell(1900-1949)
https://en.wikipedia.org/wiki/Margaret_Mitchell
https://en.wikipedia.org/wiki/Gone_with_the_Wind

随风而逝
有些东西长久地占据着它所在的位置，虽然在可以想见的未来它们不会发挥任何作用，但还是放不下太多的“如果”，不会主动丢弃
还有很多东西的建立是基于一时之兴，或出于某种臆想中的伟大，但时间略一长，便难寻踪迹
并没有起高楼，并没有不可动摇之种种，随随便便就消散不见，只有下一次的意欲起高楼
一份难能可贵的坚持，一座虽残破但绵延到视野之外的长城，金字塔，并没有

删了又删、建了又建的账号，何时能有骄傲自己的不能割舍的一张成绩单
否定，否定，否定，轻易的一次次否定，就如此的容不下你所做的哪怕毫厘之功
微不足齿，计较不得。但是你做不到永久的隐形，你或他人必须见你阳光下的影子才敢去相信，你的存在
抑或你并不存在。你只存在于你过去某一刻的空洞虚无的想象之中。
